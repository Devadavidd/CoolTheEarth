# 🌍 CoolTheEarth

An iOS application promoting environmental awareness and sustainable practices.

## Tech Stack

- **Language:** Swift
- **Framework:** SwiftUI
- **IDE:** Xcode
- **Platform:** iOS

## Features

- Environmental awareness content and education
- Interactive user interface built with SwiftUI
- Native iOS experience

## Getting Started

1. Open `CoolTheEarth.xcodeproj` in Xcode
2. Select an iOS simulator or device
3. Build and run (⌘R)

## Requirements

- Xcode 15+
- iOS 17+
- Swift 5.9+
